Admin Panel Guide

Web-based dashboard for managing users, collection points, schedules, broadcasts, and analytics. Runs on port 3001.

Accessing the Panel

# Start the admin panel
python admin_panel/app.py

# Open in browser
# http://localhost:3001

Login with the credentials from your .env file:

VariableDefaultDescription
ADMIN_PANEL_USERNAMEadminLogin username
ADMIN_PANEL_PASSWORDadminLogin password (change in production!)
ADMIN_PANEL_PORT3001HTTP port
Important: Change the default password before deploying to production. Set ADMIN_PANEL_PASSWORD in your .env to a strong, unique value.

Features Overview

Dashboard

Overview statistics: total users, active users, total points, messages, images, role distribution, and recent activity.

User Management

View all users, create new users, edit roles/points/status, delete users. Supports search and role filtering.

Collection Points

Manage waste collection locations with name, type (TPS/TPA), GPS coordinates, accepted waste types, schedule, and contact info.

Collection Schedules

Create and manage weekly pickup schedules per area with day, time range, waste types, collector name, and notes.

Broadcast

Send announcement messages to all active registered users via WhatsApp and/or Telegram channels.

Analytics

Monthly user growth, message volume, image submissions, top users by points, and activity trends.

User Roles

The system supports three roles with different privileges:

RoleDescriptionCapabilities
warga Citizen (default) Chat with bot, waste classification, view schedules/locations, earn points
koordinator Village coordinator All warga features + receive reports, view statistics via chat commands
admin System administrator All features + user management, broadcast, backup, system commands via /admin

WhatsApp/Telegram Admin Commands

Admins can also manage the system directly through chat using the /admin prefix:

CommandDescription
/admin helpShow all available admin commands
/admin statsView system statistics
/admin usersList registered users
/admin broadcast <message>Send broadcast to all active users
/admin reportGenerate and send activity report
/admin backupExport data backup
/admin memory_statsView conversation memory statistics

Security Features

  • CSRF Protection — All POST forms include a csrf_token hidden field via flask-wtf
  • Rate Limiting — Login endpoint limited to 5 attempts/minute, 20/hour
  • Session Hardening — HttpOnly, SameSite=Strict, Secure (production), 8-hour timeout
  • Constant-Time Auth — Password comparison uses hmac.compare_digest()
