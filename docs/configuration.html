Configuration

Complete reference for all environment variables. Copy .env.example to .env and fill in your values.

Application

VariableRequiredDefaultDescription
ENVIRONMENTNodevelopmentApp environment: development or production
APP_NAMENoEcoBotApplication display name
APP_VERSIONNo2.0.0Application version string
PORTNo8000API server port
TIMEZONENoAsia/JakartaTimezone for scheduling and logs
VILLAGE_NAMENo-Village name for personalized messages
VILLAGE_COORDINATESNo-GPS coordinates for collection point proximity
REGISTRATION_MODENoautoUser registration: auto (instant) or manual (admin approval)

Database

VariableRequiredDefaultDescription
DATABASE_URLYespostgresql://postgres:postgres@localhost:5432/ecobotPostgreSQL connection string
POSTGRES_USERNopostgresDB user (Docker Compose override)
POSTGRES_PASSWORDNopostgresDB password (Docker Compose override)
POSTGRES_DBNoecobotDB name (Docker Compose override)

AI Provider

VariableRequiredDefaultDescription
AI_PROVIDERNogeminiAI backend: gemini or openai
AI_API_KEYYes-API key for the AI provider
AI_MODELNogemini-2.0-flashModel name (auto-detected per provider)
AI_BASE_URLNoAutoCustom API base URL (leave blank for default)

WhatsApp (WAHA)

VariableRequiredDefaultDescription

WAHA_BASE_URLIf enabled-WAHA HTTP API base URL
WAHA_API_KEYIf enabled-WAHA authentication API key
WAHA_SESSION_NAMENodefaultWAHA session name
WAHA_WEBHOOK_SECRETIf enabled-Secret for webhook HMAC verification (fail-closed)

Security & Admin Panel

VariableRequiredDefaultDescription
API_SECRET_KEYYes-Secret for X-API-Key header validation (user API)
ADMIN_PANEL_SECRET_KEYYes-Flask session secret key for admin panel
ADMIN_PANEL_USERNAMENoadminAdmin panel login username
ADMIN_PANEL_PASSWORDYesadminAdmin panel login password (change in production!)
ADMIN_PANEL_PORTNo3001Admin panel HTTP port
Security: Never commit real API keys or passwords to version control. The _check_secrets() function in src/app.py will reject weak/placeholder secrets in production mode.
