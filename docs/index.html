EcoBot Documentation

AI-powered waste management platform for WhatsApp & Telegram. Built with Flask, PostgreSQL, and Gemini/OpenAI — version 2.0.0.

Get Started API Reference GitHub

Platform Overview

EcoBot is a Python/Flask backend that connects messaging channels (WhatsApp via WAHA, Telegram) to an AI assistant specialized in waste management education, collection scheduling, and citizen engagement.

Multi-Channel Chatbot

Receive and reply to messages on WhatsApp & Telegram with automatic AI-powered responses, waste classification from photos, and role-based commands.

Admin Panel

Web-based dashboard on port 3001 for managing users, collection points, schedules, broadcast messages, and analytics.

REST API

Programmatic access to user data, health checks, and webhook endpoints. Protected with API key authentication and HMAC-verified webhooks.

AI Vision & Text

Powered by Gemini or OpenAI for natural language understanding and image-based waste type classification with recycling guidance.

Key Features

  • AI Modes — EcoBot (database-only), General, and Hybrid modes selectable via WhatsApp commands
  • Conversation Memory — Long-term memory stored in PostgreSQL (user facts & chat history)
  • Waste Image Classification — Send a photo and get AI-powered waste type identification & recycling tips
  • Registration Modes — Auto or manual user registration, controlled by REGISTRATION_MODE
  • Role-Based Access — Warga, Coordinator, and Admin roles with different command sets
  • Collection Schedules — Manage and broadcast waste pickup schedules per area
  • Broadcast Messages — Send announcements to all active users across channels
  • Security Hardened — CSRF protection, rate limiting, HMAC webhook verification, session hardening
